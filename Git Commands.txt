1) Open git bash in project folder.

2) Initializing the project with git:- git init

3) Add the project to git :- git add .

4) commit:- git commit -m "commit name"

5) change branch master to main:- git branch -M main

6) Link with github repo:- git remote add origin https://your repo link

7) Push the project 1st time:- git push -u origin main
